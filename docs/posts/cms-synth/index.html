Source

CMS has started to provide sythetic datasets so that people can familiarize themselves working with CMS data.

On 2/12/2013 CMS released the Data Entrepreneurs’ Synthetic Public Use File (DE-SynPUF). The file contains synthesized data taken from a 5% random sample of Medicare beneficiaries in 2008 and their claims from 2008 to 2010 and preserve the detailed data structure and metadata of key variables at both the beneficiary and claim levels.

The DE-SynPUF was a joint project of NORC at the University of Chicago and IMPAQ International, a Maryland-based research, evaluation and technical assistance. All variables in the DE-SynPUF are imputed, suppressed, and coarsened as part of disclosure treatment.

To date, this is still the only release available

Dataset

Available at this CMS website

Dataset description

Contains multiple separate files.

Data documentation

Represents a 5% sample of Medicare beneficiaries.

  • Three years of claims (2008, 2009, and 2010) for the beneficiaries
  • Four types of claims (Inpatient, Outpatient, Carrier, PDE – Prescription Drug Event)
  • A beneficiary summary file for each year.

Due to file size limitations, each data type in the CMS Linkable 2008-2010 Medicare DE-SynPUF is released in 20 separate samples (essentially each is a .25% sample of all Medicare enrollees). All claims for a particular beneficiary are in samples with the same number (i.e. all beneficiaries in sample 1 have all their claims in the sample 1 files). This design allows DE-SynPUF users who do not need the entire synthetic population of the DE-SynPUF to read in only as many samples as they desire.

Example

Dataset with the entire population of benficiaries in the sample (2,326,705) and several variables.

use https://tphofer.github.io/ncsp-hhcr/data/cms_oop_costs.dta

Code creating the dataset from the 20 separate files of the beneficiary summary file for 2008 (each with about 116,000 records) is below. The files were hosted as csv files (rather than zipped files as on the CMS webstie) on a GitHub user’s site.

Code 
  clear all
  frames reset
  frame create master
  local base "https://raw.githubusercontent.com/GauriSaran/Inferential-Statistics-Project-Medicare-Claims/refs/heads/master/DE1_0_2008_Beneficiary_Summary_File_Sample"

  forvalues i = 1/20 {
      import delimited "`base'_`i'.csv", varnames(1) clear
      keep bene_sex_ident_cd sp_chf sp_cncr benres_ip benres_op benres_car
      rename bene_sex_ident_cd sex
      rename sp_chf heart_failure
      rename sp_cncr cancer
    cwf master
    fframeappend, using(default)
    cwf default
  }

  frame change master
  egen total=rowtotal(benres_ip benres_op benres_car)

Note:
benres_ip: Captures all inpatient hospitalizations, but it also includes Skilled Nursing Facility (SNF) and Hospice cost-sharing.
benres_op: Captures hospital outpatient departments, ER visits, and Home Health Agency (HHA) institutional outpatient claims.
benres_car: Captures non-institutional professional services, such as doctor office visits, lab tests, and Durable Medical Equipment (DME).
